## FinHub — Fintech Dashboard

React + Vite + Tailwind CSS дашборд‑агрегатор со стеком:

- React + TypeScript (Vite)
- Tailwind CSS (glassmorphism‑дизайн)
- Recharts (простая аналитика трат)

### Быстрый старт

```bash
cd "smartpath-dashboard"
npm install
npm run dev
```

После запуска откройте `http://localhost:5173` в браузере.
